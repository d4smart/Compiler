### 词法分析器

* 通用数字（12.13e-1）的状态转移图

![数字的状态转移图](Static/number.png)

* 正负号和加减运算符的区分

当且仅当符号+|-的后一位为数字时，符号可能为正负号； 如果数字前一个字符串具有可被加减的性质，则是运算符加减；否则是正负号
